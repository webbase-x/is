# โครงสร้างไฟล์แผนการสอน ป.1

ใช้รูปแบบต่อไปนี้สำหรับทุกแผน:

```text
P1/plan/{เลขหน่วย 2 หลัก}/แผนที่{ลำดับแผน}.docx
```

ตัวอย่าง:

```text
P1/plan/01/แผนที่1.docx
P1/plan/02/แผนที่1.docx
P1/plan/02/แผนที่2.docx
P1/plan/12/แผนที่1.docx
```

- `P1` หมายถึง ชั้นประถมศึกษาปีที่ 1
- `plan` หมายถึง แผนการสอน
- `01` ถึง `12` หมายถึง หมายเลขหน่วย
- `แผนที่1`, `แผนที่2`, ... หมายถึง ลำดับแผนการสอนภายในหน่วย
