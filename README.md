# เพลินภาษา — ศูนย์รวมสื่อการเรียนรู้ภาษาไทย ป.1–2

เว็บไซต์หลักสำหรับรวบรวมบทเรียนภาษาไทยระดับชั้นประถมศึกษาปีที่ 1–2 จัดทำโดย **ครูไพรัช อินควรชุม โรงเรียนเทศบาล ๑ (ถนนนครนอก)**

## โครงสร้างเว็บไซต์

- `/` หน้าเว็บหลัก “เพลินภาษา” และคลังบทเรียน ป.1–2
- `/P2/` ระบบเดิม “คำไทยผจญภัย” สำหรับนักเรียน ป.2 เรื่องมาตราตัวสะกด
- `/P2/js/`, `/P2/sounds/`, `/P2/supabase/` สคริปต์ เสียง และฐานข้อมูลของบทเรียน ป.2

## ระบบบทเรียน ป.2

## ความสามารถในแผนที่ 1

- ห้องรอสำหรับเด็กประถมต้น: กรอกรหัสห้อง → เลือกชื่อ → ถ่ายภาพสดและส่งทันที → รอครูอนุมัติ
- หน้าครูสร้าง QR และลิงก์เข้าห้องให้อัตโนมัติ นักเรียนสแกนแล้วข้ามการกรอกรหัสได้
- หน้าครูทำงานทีละขั้นเต็มจอ: เลือกห้อง → QR → รับนักเรียน → เลือกแผน → ผลสด → สรุปผล
- เมื่อ Login แล้วพบคาบที่ยังไม่จบ ระบบจะแสดงหน้าถามก่อนว่าจะกลับไปสอน ดูสรุป หรือปิดคาบเดิม โดยไม่ข้ามไปหน้าผลสดเอง
- ครูเห็นเฉพาะโรงเรียนและห้องที่ได้รับมอบหมาย รหัสทุกชุดผูกกับห้อง คาบ และบัญชีครู
- รหัสห้อง 6 หลักมีข้อห้ามซ้ำในฐานข้อมูล และสุ่มใหม่อัตโนมัติหากครูหลายคนสร้างพร้อมกัน
- ห้องเรียนเดียวกันเปิดได้ครั้งละหนึ่งคาบ ป้องกันครูสองคนสร้างคนละรหัสให้เด็กห้องเดียวกัน
- ห้องที่ยังไม่มีรายชื่อนักเรียนจะสร้างรหัสและ QR ไม่ได้ หน้าครูจะแสดงจำนวนรายชื่อก่อนเปิดคาบ
- รายชื่อที่ไม่ซ้ำจะแสดงเฉพาะชื่อ หากชื่อซ้ำจึงแสดงชื่อเล่นหรือรหัสเพื่อแยกนักเรียน
- รับเฉพาะภาพจากกล้องสด ไม่มีการเลือกรูปเก่าจากเครื่อง
- ครูอนุมัติรายคน อนุมัติทั้งหมด ส่งคืนพร้อมเหตุผล หรือนำออกจากห้อง
- ตั้งค่ารายคาบ: รอบฝึก/ทดสอบ/จริง จำนวนครั้ง นโยบายคะแนน เกณฑ์ผ่าน และรูปแบบกระดานคะแนน
- เกมแท็ปจังหวะ วงล้อ นักสืบเสียง จัดบ้านให้คำ รถไฟประโยค บอร์ดโหวต และ Exit Ticket
- คะแนนครั้งแรก คะแนนดีที่สุด ประวัติทุกครั้ง และส่งออก CSV
- รูปเซลฟี่อยู่ในพื้นที่ส่วนตัวและลบเมื่อครูปิดคาบ
- เตรียมไทม์ไลน์ไว้สำหรับแผนการเรียนรู้ 8 แผน

## ติดตั้งฐานข้อมูล Supabase

1. เปิด Supabase Dashboard ของโครงการ
2. ไปที่ **SQL Editor** แล้วรันไฟล์ `P2/supabase/schema.sql`
3. ไปที่ **Authentication → Providers → Anonymous Sign-Ins** แล้วเปิดใช้งาน
4. ไปที่ **Authentication → Users** และสร้างผู้ใช้ครูด้วยอีเมลและรหัสผ่าน
5. กลับไปที่ SQL Editor แล้วให้สิทธิ์ครู โดยเปลี่ยนอีเมลและชื่อในคำสั่งนี้

```sql
select public.grant_teacher(
  'teacher@example.com',
  'ชื่อครูผู้สอน',
  'admin'
);
```

บัญชีครูคนแรกควรเป็น `admin` เพื่อสร้างโรงเรียนและห้อง ป.1/1–ป.2/4 ได้ หลังจากนั้นสามารถสร้างบัญชีครูแบบ `teacher` และกำหนดห้องรับผิดชอบในตาราง `teacher_class_assignments`

หากติดตั้งฐานข้อมูลเดิมไว้แล้ว ให้รัน `P2/supabase/teacher-flow-upgrade.sql` เพิ่มอีกหนึ่งครั้ง เพื่อเปิดรายการห้องตามสิทธิ์และเสริมการป้องกันรหัสซ้ำเมื่อสร้างพร้อมกัน

จากนั้นรัน `P2/supabase/empty-class-guard.sql` หนึ่งครั้ง เพื่อป้องกันการเปิดคาบสำหรับห้องที่ยังไม่มีรายชื่อนักเรียน

## ตั้งค่าการกู้รหัสผ่านครู

1. ไปที่ **Authentication → URL Configuration** ใน Supabase Dashboard
2. ตั้ง **Site URL** เป็น `https://webbase-x.github.io/is/P2/`
3. เพิ่ม **Redirect URL** เป็น `https://webbase-x.github.io/is/P2/reset-password.html`
4. ครูสามารถกด **ลืมรหัสผ่าน?** ที่หน้าเข้าสู่ระบบ เพื่อรับอีเมลและตั้งรหัสใหม่ได้

## เตรียมรายชื่อนักเรียน

ครูเพิ่มรายชื่อทีละคน หรือนำเข้าไฟล์ Excel/CSV ได้ โดยใช้หัวคอลัมน์ต่อไปนี้:

```text
class,student_code,full_name,nickname
ป.1/1,001,เด็กชายตัวอย่าง ใจดี,น้องเอ
```

## เผยแพร่ด้วย GitHub Pages

1. นำไฟล์ทั้งหมดขึ้น branch `main`
2. เปิด **Settings → Pages**
3. เลือก **Deploy from a branch**
4. เลือก branch `main` และโฟลเดอร์ `/ (root)`

เว็บไซต์ไม่มีขั้นตอน build และสามารถเปิด `index.html` ได้โดยตรงผ่านเว็บเซิร์ฟเวอร์แบบ static

## ความปลอดภัย

- ใช้เฉพาะ Supabase public/anon key ในฝั่งเว็บ
- ห้ามใส่ `service_role` key ใน repository หรือ JavaScript ฝั่งผู้ใช้
- ฟอร์ม Login ไม่ส่งอีเมลหรือรหัสผ่านผ่าน URL และจะล้างพารามิเตอร์ Login เก่าจากแถบที่อยู่อัตโนมัติ
- ตารางทุกชุดใช้ Row Level Security (RLS)
- รายชื่อนักเรียนเปิดผ่านรหัสห้องชั่วคราวหรือ QR ของคาบนั้นเท่านั้น
- จอฉายแสดงชื่อในรูปแบบที่ครูเลือกสำหรับคาบนั้น
